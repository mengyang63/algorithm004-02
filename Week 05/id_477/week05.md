# 动态规划和递归、分治没有本质区别

1. 共性：找重复子问题， 动态规划也是将问题划分为子问题进行解决
2. 分治需要对每个子问题求解
3. 动态规划可以找最优子结构，中途淘汰次优解
4. 傻递归、啥分治，动态规划根据中间状态可以跳过一些计算

# 思路

1. 找到最优子结构
2. 推导出 DP 方程
3. 记录中间状态

# 动态规划
1. 打破思维习惯，形成机器思维（if else, for loop, 递归, 分治)
2. 理解复杂问题的关键，找到公共子问题
3. 也是职业进阶的关键(不要亲历亲为，反馈，允许犯错)
